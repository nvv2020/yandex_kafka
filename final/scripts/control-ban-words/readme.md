# Описание

## Основной обработчик process_shop_messages:

Читает сообщения из shop-raw

Фильтрует текст с использованием текущего списка запрещенных слов

Отправляет отфильтрованные сообщения в shop-clear

## Функция фильтрации filter_text:

Использует регулярные выражения для точного поиска слов

Заменяет найденные слова на звездочки

Возвращает детальную статистику фильтрации

## Обработчик запрещенных слов handle_ban_words_updates:

Слушает обновления из топика ban-words

Обновляет таблицу запрещенных слов в реальном времени

## Структура сообщений:

ShopMessage - входное сообщение из shop_raw

FilteredShopMessage - выходное сообщение в shop_clear с полной информацией о фильтрации

## API для мониторинга:

Статистика обработки

Просмотр текущего списка запрещенных слов

## Тестирование фильтрации

Генератор отправляет события ADD/REMOVE в топик ban-words

Этот сервис обновляет свой список запрещенных слов

Сообщения из shop_raw фильтруются с использованием этого списка

Отфильтрованные сообщения отправляются в shop_clear
